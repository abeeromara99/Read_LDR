# Read_LDR  
-This project utilizes Atmega32 microcontroller,LDR sensor,LCD (16x2)character.

# Driver Implemented:  
1-DIO driver
2-LCD driver  
3-ADC driver  

# How it works ?  
-if there is light Hamoksha will wake up and say Good Morning.  
-if there is dark Hamoksha will sleep and say Good Evening.
